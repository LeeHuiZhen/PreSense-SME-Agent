# ⬡ PreSense: SME Autonomous Maintenance Agent
### V HACK 2026 — ASEAN: Inclusivity & Sustainability in the Digital Era
 
> **Transforming ASEAN's SME factories from reactive "break-fix" to proactive predictive maintenance — powered by ML, MCP, and Agentic AI.**
 
---
 
## 🏭 Problem Statement
 
Small and medium enterprises (SMEs) across ASEAN operate on a reactive maintenance model — machines are only serviced after they break down. This causes:
 
- **Unplanned downtime** averaging RM 4,200/hour in lost production
- **Emergency part procurement** at 3–5× standard cost
- **Safety risks** from sudden equipment failure
- **Inability to plan** workforce and inventory efficiently
 
PreSense solves this by predicting equipment failure **before it happens**.
 
---
 
## 💡 Solution Overview
 
PreSense is a three-layer autonomous maintenance platform:
 
```
┌─────────────────────────────────────────────────────────┐
│  LAYER 1: ML ENGINE                                     │
│  Random Forest Regressor trained on NASA C-MAPSS        │
│  Predicts Remaining Useful Life (RUL) from 21 sensors   │
│  RMSE: 19.00 | S-Score: 1207.93 | NASA FD001 benchmark  │
├─────────────────────────────────────────────────────────┤
│  LAYER 2: AGENTIC DECISION INTELLIGENCE                 │
│  Multi-factor Risk Score Engine                         │
│  PELT Change-Point Detection (ruptures library)         │
│  SHAP Explainability — shows WHY the model predicts     │
├─────────────────────────────────────────────────────────┤
│  LAYER 3: MCP AUTONOMOUS DISPATCH                       │
│  Real FastMCP server with 3 registered tools            │
│  check_inventory()      → query spare parts             │
│  schedule_technician()  → assign available technician   │
│  create_work_order()    → auto-dispatch maintenance     │
└─────────────────────────────────────────────────────────┘
```
 
---
 
## 🤖 AI Technology Stack
 
| Component | Technology | Purpose |
|---|---|---|
| RUL Prediction | Random Forest Regressor | Predict cycles until failure |
| Feature Engineering | Rolling Mean (window=3) | Smooth sensor noise |
| Explainability | SHAP TreeExplainer | Show top degradation drivers |
| Change Detection | PELT Algorithm (ruptures) | Find regime shift cycle |
| Agentic Layer | Risk Score Engine | Multi-factor AUTO_DISPATCH decision |
| MCP Server | FastMCP | Real tool-calling protocol |
| Dashboard | Streamlit | Cyber-industrial real-time UI |

## 📊 Model Performance
 
Trained and evaluated on the **NASA C-MAPSS Turbofan Engine Dataset (FD001)**:
 
| Metric | Baseline | PreSense v2.0 |
|---|---|---|
| Test RMSE | 41.66 | **19.00** |
| Improvement | — | **+54.4%** |
| S-Score (official) | — | **1207.93** |
| Features Used | 21 raw sensors | 28 (14 sensors + 14 rolling) |
| RUL Labelling | Linear decay | Piecewise capped at 125 |
 
> PreSense achieves RMSE competitive with LSTM deep learning models
> while remaining interpretable and deployable without a GPU.

## 🚀 Quick Start
 
### 1. Install Dependencies
```bash
pip install -r requirements.txt
```
 
### 2. Train the Model
```bash
python train.py
```
 
Expected output:
```
OFFICIAL TEST RMSE : 19.00  (was 41.66)
IMPROVEMENT        : +54.4%
OFFICIAL S-SCORE   : 1207.93  (lower is better)
```
 
### 3. Run the Dashboard
```bash
streamlit run app.py
```
 
### 4. (Optional) Test MCP Server Standalone
```bash
python mcp_server.py
```
